# 🤖 DiscountMart Bot

A simple Discord bot built for the DiscountMart digital gaming store.

**🚀 Features**
- 📦 Inventory System
Shows available products and their quantity
- 💬 Message Triggers
Automatically replies to specific messages
- ℹ️ About Command
Displays basic information about the bot

**🛠️ Tech Stack**
- Python
- discord.py

**📂 Project Structure**
discountmartbot/
│
├── bot.py             # Main bot file
├── .env               # Environment variables (token, etc.)
└── requirements.txt   # Dependencies

**⚙️ Setup & Installation**
**1. Clone the repository**
git clone https://github.com/FlashTanmay-YT/discountmartbot.git
cd discountmartbot

**2. Install dependencies**
pip install -r requirements.txt

**3. Setup environment variables**
Create a .env file:
TOKEN=your_discord_bot_token

**4. Run the bot**
python bot.py

**📌 Requirements**
- Python 3.8+
- A Discord Bot Token

**🔐 Note**
Do not share your .env file or bot token publicly.
